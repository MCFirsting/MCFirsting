Resource Packs เบื้องต้น

เปลี่ยนหน้าตาโลก Minecraft ของคุณด้วยการปรับแต่ง Texture ง่ายๆ

ก่อนที่คุณจะเริ่มสร้าง Add-On ที่ซับซ้อน สิ่งแรกที่ต้องทำความเข้าใจคือการสร้าง "Pack" เพื่อเก็บข้อมูลของคุณ ใน Minecraft: Bedrock Edition จะมี 2 ประเภทหลักคือ Resource Packs และ Behavior Packs

Resource Pack คือโครงสร้างโฟลเดอร์ที่เก็บไฟล์ปรับแต่งรูปลักษณ์ทั้งหมด เช่น โมเดล (Models), เสียง (Sounds), และพื้นผิว (Textures) ในบทเรียนนี้เราจะมาลองเปลี่ยนลายของบล็อกหญ้ากัน!

โครงสร้างไฟล์และโฟลเดอร์

เพื่อให้ Minecraft หาไฟล์ของคุณเจอ เราต้องสร้างโฟลเดอร์และไฟล์ตามโครงสร้างที่กำหนด เริ่มจากเข้าไปที่โฟลเดอร์ com.mojang (ดูวิธีเข้าได้ที่หน้า พื้นฐานการเริ่มต้น)

โครงสร้างไฟล์ของคุณควรจะเป็นแบบนี้:

1. สร้างโฟลเดอร์ Pack

เข้าไปที่โฟลเดอร์ development_resource_packs แล้วสร้างโฟลเดอร์ใหม่ตั้งชื่อว่า: 

My_RESOURCE_Pack

*คุณสามารถตั้งชื่ออื่นได้ แต่ในบทเรียนนี้จะใช้ชื่อนี้เพื่อไม่งงครับ

2. สร้างไฟล์ Manifest

ไฟล์ manifest.json เปรียบเสมือนบัตรประชาชนของ Pack ที่บอกเกมว่า Pack นี้ชื่ออะไร เวอร์ชั่นไหน ให้สร้างไฟล์ text ขึ้นมาในโฟลเดอร์ My_RESOURCE_Pack แล้วเปลี่ยนชื่อเป็น manifest.json

จากนั้นเปิดไฟล์และวางโค้ดนี้ลงไป:

{
            "format_version": 2,
            "header": {
                "description": "My dirt resource pack Add-On!",
                "name": "My Resource Pack",
                "uuid": "<วาง UUID ที่ 1 ตรงนี้>",
                "version": [1, 0, 0],
                "min_engine_version": [1, 16, 0]
            },
            "modules": [
                {
                    "description": "My First Add-On!",
                    "type": "resources",
                    "uuid": "<วาง UUID ที่ 2 ตรงนี้>",
                    "version": [1, 0, 0]
                }
            ]
        }

สำคัญ: เรื่อง UUID

UUID คือรหัสตัวเลขที่ไม่ซ้ำกัน คุณต้องสร้างรหัสใหม่ 2 รหัสจากเว็บ uuidgenerator.net แล้วนำมาวางแทนที่ <วาง UUID ...> ในโค้ดด้านบน (ต้องใช้คนละรหัสกันระหว่าง Header และ Modules)

เปลี่ยนลายบล็อกดิน Dirt Block

เมื่อเตรียมไฟล์ Manifest เสร็จแล้ว เรามาเริ่มใส่รูปภาพกันครับ

  1. ในโฟลเดอร์ My_RESOURCE_Pack ให้สร้างโฟลเดอร์ชื่อ textures
  2. เข้าไปใน textures แล้วสร้างโฟลเดอร์ชื่อ blocks
  3. สร้างรูปภาพขนาด 16x16 พิกเซล (ไฟล์ .png)
  4. ตั้งชื่อรูปภาพว่า dirt.png แล้วนำไปวางในโฟลเดอร์ blocks ที่เพิ่งสร้าง

(สำหรับบทเรียนนี้คุณสามารถบันทึกรูปตัวอย่างได้ครับ)

ทดสอบในเกม

เมื่อทำทุกอย่างครบแล้ว ให้เปิดเกม Minecraft ขึ้นมา:

  • กด Play > Create New World
  • เลื่อนลงมาที่แท็บ Resource Packs ทางซ้ายมือ
  • ในส่วน Available > Owned คุณจะเห็นชื่อ "My Resource Pack" ที่เราสร้างไว้
  • กด Activate แล้วสร้างโลกได้เลย!

ถ้าทำสำเร็จ บล็อกดินในเกมจะเปลี่ยนเป็นรูปที่คุณวาดไว้ (ยกเว้นดินที่มีหญ้าปกคลุมนะ!)

ปัญหาที่พบบ่อย

เจอปัญหา Resource pack ใช่ไหม? ไม่ต้องห่วง การแก้ไขปัญหาเป็นเรื่องปกติของการพัฒนาครับ

คุณสามารถดาวน์โหลด ตัวอย่าง Resource Pack จาก GitHub ของ Minecraft มาเช็คโครงสร้างไฟล์เทียบกับของคุณได้

Resource Pack ไม่ขึ้นในเกม

ถ้า Pack ของคุณไม่โผล่ในหน้าตั้งค่า Add-ons ของโลก อาจเกิดจากไฟล์ manifest.json ผิดพลาด ลองเช็คตามนี้:

  • UUID ซ้ำกันไหม?: เช็คว่า UUID ในส่วน header และ modules เป็นคนละตัวกันหรือไม่ (ดูหัวข้อ UUID ด้านบน)
  • นามสกุลไฟล์ถูกต้องไหม?: Windows ของคุณโชว์นามสกุลไฟล์หรือยัง? ถ้ายัง ให้ไปเปิด 'File Name Extensions' ในแถบ View ของ File Explorer เพื่อให้แน่ใจว่าเป็น .json ไม่ใช่ .json.txt
  • วงเล็บครบไหม?: เช็ควงเล็บปีกกา { } และก้ามปู [ ] ให้ดีว่าเปิดปิดครบถ้วน (แนะนำให้ใช้ VS Code จะช่วยเตือนจุดผิดได้)

Resource Pack ขึ้น แต่รูปไม่เปลี่ยน (Content ไม่ทำงาน)

  • เช็คชื่อและที่อยู่โฟลเดอร์: ตรวจสอบว่าโฟลเดอร์ textures และ blocks สะกดถูกเป๊ะๆ และวางซ้อนกันถูกลำดับ
  • ชื่อไฟล์รูปภาพ: แน่ใจนะว่าตั้งชื่อไฟล์รูปเหมือนกับไฟล์ต้นฉบับเป๊ะๆ (เช่น dirt.png) หรือไม่
  • ลำดับของ Pack: ลองเลื่อน Pack ของคุณไปไว้บนสุดในหน้าตั้งค่าเกม เพื่อให้มันทำงานทับ Pack อื่นๆ
