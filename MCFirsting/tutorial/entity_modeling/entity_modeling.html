การสร้างโมเดลและอนิเมชัน Entity Modeling

เปลี่ยนจินตนาการให้เป็น 3D และทำให้มันเคลื่อนไหวด้วย Blockbench

ในบทนี้เราจะมาลงลึกเรื่อง "รูปลักษณ์ภายนอก" กันครับ เราจะสร้างโมเดลหุ่นยนต์ (Robot) ลงสี และใส่ท่าทางให้มัน โดยใช้โปรแกรมยอดนิยมอย่าง Blockbench

สิ่งที่คุณต้องเตรียม

  • ความเข้าใจเรื่อง Resource Pack และ Behavior Pack (จากบทเรียนก่อนหน้า)

    • เตรียมเครื่องมือ

    เราจะใช้โปรแกรม Blockbench ซึ่งเป็นเครื่องมือสร้างโมเดล 3D สำหรับ Minecraft ที่ดีที่สุดในตอนนี้ (ฟรีและใช้งานง่ายมาก)

    • ดาวน์โหลด Blockbench (หรือใช้ผ่านเว็บก็ได้)

    ส่วนที่ 1: การสร้างโมเดล (Modeling)

    1. ตั้งค่าโปรเจกต์

    เมื่อเปิด Blockbench ให้เลือกเมนู Bedrock Model จากนั้นตั้งค่าดังนี้:

    • File Name: robot
    • Model Identifier: geometry.robot (สำคัญมาก! ต้องตรงกับไฟล์ JSON ของเรา)
    • Texture Size: 64x64 (หรือ 32x32 สำหรับโมเดลเล็กๆ)

    2. สร้างโครงกระดูก

    โครงสร้างกระดูก (Bone) คือหัวใจสำคัญของการทำอนิเมชัน กระดูกแต่ละชิ้นจะทำหน้าที่เหมือน "ข้อต่อ" ที่เราสามารถหมุนหรือขยับได้

    เทคนิค: ควรสร้างกระดูกหลัก (Root Bone) หนึ่งชิ้นเพื่อคุมทั้งตัว แล้วค่อยสร้างกระดูกย่อย (เช่น หัว, แขน, ขา) ใส่เข้าไปข้างใน จะทำให้จัดการท่าทางได้ง่ายขึ้น

    ตัวอย่างโครงสร้างกระดูกของหุ่นยนต์:

    root (กระดูกหลัก)
 ├── head (หัว)
 ├── body (ลำตัว)
 ├── arm_right (แขนขวา)
 ├── arm_left (แขนซ้าย)
 └── wheel (ล้อ - สำหรับอนิเมชันขับเคลื่อน)

    3. ปั้นทรง (Cube)

    ใช้เครื่องมือ Cube สร้างสี่เหลี่ยมขึ้นมาแล้วจัดวางให้เป็นรูปร่าง อย่าลืมลาก Cube ไปใส่ใน Bone ที่ถูกต้องด้วยนะครับ (เช่น ลูกตาก็ต้องอยู่ใน Bone หัว)

    ส่วนที่ 2: การลงสี (Texturing)

    เมื่อปั้นเสร็จแล้ว ให้กด Create Texture ใน Blockbench (เลือกแบบ Template เพื่อให้โปรแกรมกาง UV ให้เอง) จากนั้นไปที่แท็บ Paint เพื่อระบายสีลงบนโมเดล 3D ได้เลย!

    เมื่อเสร็จแล้วให้บันทึกไฟล์:

    • Model: robot.geo.json (นำไปใส่ในโฟลเดอร์ models/entity/)
    • Texture: robot.png (นำไปใส่ในโฟลเดอร์ textures/entity/)

    ส่วนที่ 3: การทำอนิเมชัน (Animation)

    กลับมาที่ Blockbench เปลี่ยนไปที่แท็บ Animate แล้วกดเครื่องหมาย + เพื่อสร้างอนิเมชันใหม่ ตั้งชื่อว่า animation.robot.sway (ท่าโยกตัว)

    ขั้นตอนการทำ Keyframe

    1. เลือก Bone ที่ต้องการขยับ (เช่น body)
    2. เลื่อนแถบเวลาไปที่ 0 วินาที -> กดปุ่ม Rotation แล้วขยับเล็กน้อยเพื่อสร้างจุดเริ่มต้น
    3. เลื่อนไปที่ 0.5 วินาที -> หมุนตัวไปทางซ้าย
    4. เลื่อนไปที่ 1.0 วินาที -> หมุนตัวไปทางขวา
    5. เลื่อนไปที่ 1.5 วินาที -> หมุนกลับมาตรงกลาง
    6. ตั้งค่า Loop เป็น True เพื่อให้เล่นวนซ้ำ

    เมื่อเสร็จแล้ว ให้บันทึกไฟล์อนิเมชันเป็น robot.animation.json แล้วนำไปใส่ในโฟลเดอร์ animations/ ของ Resource Pack

    ส่วนที่ 4: เชื่อมต่อในโค้ด

    สุดท้าย เราต้องบอกเกมให้เล่นอนิเมชันนี้ โดยไปแก้ไฟล์ Client Entity (robot.entity.json): 

    "animations": {
    "sway": "animation.robot.sway"
},
"scripts": {
    "animate": [
        { "sway": "!query.is_on_ground" }
    ]
}

    อธิบายโค้ด: เราตั้งเงื่อนไขว่า !query.is_on_ground (เมื่อไม่ได้อยู่บนพื้น) ให้เล่นท่า sway ดังนั้นเวลาหุ่นยนต์ลอยตัวหรือตกจากที่สูง มันจะโยกตัวไปมาครับ!

    บทสรุป

    ตอนนี้คุณมีหุ่นยนต์ที่มีโมเดลเป็นของตัวเอง มีสีสัน และขยับได้แล้ว! การฝึกฝนใช้ Blockbench บ่อยๆ จะช่วยให้คุณสร้างสรรค์ Entity ที่ซับซ้อนและสวยงามขึ้นได้เรื่อยๆ ครับ
