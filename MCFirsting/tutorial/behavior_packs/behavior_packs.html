Behavior Packs เบื้องต้น

เปลี่ยนพฤติกรรมของสิ่งมีชีวิตและกฎเกณฑ์ในโลก Minecraft

Behavior Pack คือชุดไฟล์ที่กำหนด "นิสัย" หรือ "พฤติกรรม" ของสิ่งต่างๆ ในเกม เช่น ทำให้วัวโจมตีผู้เล่น, เปลี่ยนของที่ดรอปจากซอมบี้, หรือแม้แต่สร้างสูตรคราฟต์ของขึ้นมาใหม่

โครงสร้างไฟล์และโฟลเดอร์

เพื่อให้ Minecraft หาไฟล์ของคุณเจอ เราต้องสร้างโฟลเดอร์และไฟล์ตามโครงสร้างที่กำหนด เริ่มจากเข้าไปที่โฟลเดอร์ com.mojang (ดูวิธีเข้าได้ที่หน้า พื้นฐานการเริ่มต้น)

โครงสร้างไฟล์ของคุณควรจะเป็นแบบนี้:

1. สร้างโฟลเดอร์ Pack

สร้างโฟลเดอร์ใหม่ตั้งชื่อว่า My_BEHAVIOR_Pack

2. สร้างไฟล์ Manifest

สร้างไฟล์ manifest.json ในโฟลเดอร์นั้น แล้วใส่โค้ดด้านล่าง (สังเกตว่าตรง type จะเป็น data แทน resources) 

{
    "format_version": 2,
    "header": {
        "description": "My Angry Cow Add-On",
        "name": "My Behavior Pack",
        "uuid": "<UUID ที่ 1>",
        "version": [1, 0, 0],
        "min_engine_version": [1, 16, 0]
    },
    "modules": [
        {
            "description": "My First Behavior Mod",
            "type": "data",
            "uuid": "<UUID ที่ 2>",
            "version": [1, 0, 0]
        }
    ]
}

อย่าลืมเปลี่ยน <UUID> ทั้ง 2 จุดด้วยรหัสจาก uuidgenerator.net นะครับ

ทดสอบในเกม

เมื่อทำทุกอย่างครบแล้ว ให้เปิดเกม Minecraft ขึ้นมา:

  • กด Play > Create New World
  • เลื่อนลงมาที่แท็บ Behavior Packs ทางซ้ายมือ
  • ในส่วน Available > Owned คุณจะเห็นชื่อ "My Behavior Pack" ที่เราสร้างไว้
  • กด Activate แล้วสร้างโลกได้เลย!

ถ้าทำสำเร็จ ระวังตัวให้ดี เพราะมันจะไม่ใช่วัวที่แสนเชื่องอีกต่อไป!

ปัญหาที่พบบ่อย

เจอปัญหา Behavior pack ใช่ไหม? ไม่ต้องกังวล การแก้ไขปัญหาเป็นเรื่องปกติของการพัฒนาครับ นี่คือจุดเริ่มต้นในการตรวจสอบ:

  • เช็ค Content Error Logs: ตรวจสอบ Content Error Logs ในเกมเพื่อดูแจ้งเตือนข้อผิดพลาดที่อาจทำให้เกมไม่เสถียรหรือทำงานผิดปกติ
  • ตรวจสอบโครงสร้างไฟล์: ดาวน์โหลด ตัวอย่าง Behavior Pack จาก GitHub เพื่อนำมาเปรียบเทียบโครงสร้างไฟล์กับของคุณว่าถูกต้องหรือไม่
