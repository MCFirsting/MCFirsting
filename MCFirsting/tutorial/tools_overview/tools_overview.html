รู้จักกับเครื่องมือที่จำเป็น

เตรียม "อาวุธ" ให้พร้อมก่อนลงสนามสร้าง Add-ons

การสร้าง Minecraft Add-ons ไม่ใช่แค่การเล่นเกม แต่คือการ "พัฒนาซอฟต์แวร์" รูปแบบหนึ่งครับ ดังนั้นเราจึงต้องการเครื่องมือเฉพาะทางที่จะช่วยให้เราเขียนโค้ด วาดรูป และปั้นโมเดลได้อย่างราบรื่น นี่คือรายการ "ของมันต้องมี" สำหรับนักพัฒนา Add-ons มือใหม่ครับ

เครื่องมือยอดนิยม

มีเครื่องมือหลากหลายประเภทที่ถูกสร้างขึ้นเพื่อจุดประสงค์ในการเพิ่มประสิทธิภาพการสร้างสรรค์เนื้อหาสำหรับ Minecraft เครื่องมือเหล่านี้สามารถช่วยงานได้หลากหลายด้าน ตั้งแต่การสร้างสกินไปจนถึงการเจาะระบบเกม เพื่อเพิ่มฟีเจอร์ใหม่ๆ ให้กับเกม

ในบทความนี้ เราจะมาสำรวจเครื่องมือที่นิยมใช้กันมากที่สุดในปัจจุบันและดูว่าพวกมันมีประโยชน์อย่างไรบ้าง

⚠️ ข้อสำคัญ (Important)

เครื่องมือเหล่านี้จัดทำโดยผู้พัฒนาภายนอก (Third-party contributors) และ ไม่ใช่ ผลิตภัณฑ์ของ Mojang หรือ Microsoft โดยตรง ผู้ใช้งานควรใช้งานโดยใช้วิจารณญาณของตนเองครับ

การสร้างโมเดลและภาพประกอบ

Blockbench

Blockbench คือแอปพลิเคชันสำหรับปั้นโมเดล Entity แบบฟรี ที่ช่วยให้คุณสร้างโมเดลสำหรับ Bedrock Edition ที่ใช้งานได้จริง รวมถึงการสร้าง Texture และทำอนิเมชันสำหรับแมพใน Bedrock Edition ครับ

Blockbench รองรับการใช้งานปลั๊กอินภายนอก ซึ่งเปิดโอกาสให้นักพัฒนาสามารถสร้างเครื่องมือเสริมเพื่อใช้งานภายในโปรแกรมได้ นอกจากนี้ยังสามารถส่งออกโมเดล (Export) เป็นไฟล์รูปแบบมาตรฐาน OBJ, โมเดลบล็อกของ Java Edition และรูปแบบอื่นๆ ได้อีกมากมาย

Snowstorm

Snowstorm เป็นแอปพลิเคชันขนาดเล็กสำหรับ Bedrock Edition ที่สร้างขึ้นโดยผู้พัฒนาเดียวกับ Blockbench ซึ่งใช้สำหรับสร้างเอฟเฟกต์อนุภาค (Particle effects) แบบกำหนดเองเพื่อนำไปใช้ในเกมครับ

หากต้องการเริ่มต้นใช้งาน Snowstorm สามารถเข้าไปศึกษาได้ที่บทช่วยสอนการสร้าง Particle effects ด้วย Snowstorm ครับ

การแก้ไขโปรเจกต์และไฟล์ JSON

Blockception VSCode Extension

เป็นส่วนขยาย (Extension) สำหรับโปรแกรม Visual Studio Code ที่ให้การสนับสนุนไฟล์ต่างๆ เช่น .mcfunction, .json และ .lang ครับ

ส่วนขยายนี้เพิ่มฟีเจอร์ที่มีประโยชน์มากมาย เช่น การเติมคำอัตโนมัติ (Completion), การตรวจสอบความถูกต้อง (Validation), การจัดรูปแบบโค้ด (Formatter), การวิเคราะห์ปัญหา (Diagnostics), โพยคำสั่ง (Cheat-sheets), การกระทำกับโค้ด (Code-actions), การสร้างไฟล์, และเครื่องมือพัฒนาอื่นๆ ที่ช่วยให้การสร้าง Add-ons สำหรับ Minecraft: Bedrock Edition หรือ Minecraft Education ง่ายขึ้นมากครับ

แนะนำให้ใช้ธีม Dark+ เพื่อให้การแสดงสีไฮไลท์ของโค้ดทำงานได้ดีที่สุดครับ

Bridge. V2

Bridge. V2 เป็นโปรแกรมแก้ไข (Editor) ขนาดเล็กที่ออกแบบมาเพื่อ Minecraft Add-ons โดยเฉพาะ ช่วยให้เริ่มต้นทำ Add-ons ได้ง่าย และรองรับความต้องการที่ซับซ้อนขึ้นในอนาคต

จุดเด่นคือ ไม่ต้องตั้งค่าใดๆ เปิดมาก็พร้อมใช้งานทันที มาพร้อมสภาพแวดล้อมการพัฒนาที่มีทั้งการไฮไลท์ไวยากรณ์ (Syntax highlighting), การเติมโค้ดอัตโนมัติ, และการวิเคราะห์ไฟล์ ซึ่งทั้งหมดออกแบบมาเพื่อความสะดวกในการพัฒนา Add-ons ครับ

คุณสามารถอ่านเพิ่มเติมได้ที่คู่มือการเริ่มต้นใช้งาน Bridge. V2 หรือเรียนรู้ เหตุผลว่าทำไมคุณถึงควรใช้ Bridge. V2 โดยโปรแกรมนี้เป็น Open Source และเปิดให้ครีเอเตอร์ทุกคนใช้งานได้ฟรีทั้งบนคอมพิวเตอร์และมือถือครับ

Minecraft Creator Tools

Minecraft Creator Tools เป็นทั้งเว็บไซต์และเครื่องมือ Command line (บน NPM) ที่คุณสามารถใช้เพื่อเริ่มต้นหรือขยายโปรเจกต์ Minecraft ของคุณ นอกจากนี้ยังมีตัวตรวจสอบ (Validators) เพื่อช่วยค้นหาและชี้จุดที่อาจเกิดปัญหาในโปรเจกต์ได้อีกด้วย

เครื่องมือนี้เป็น Open Source เช่นกันและสามารถเข้าไปดูได้บน GitHub ครับ

NPC DialogueDesigner

DialogueDesigner เป็นเว็บแอปพลิเคชันสำหรับ Bedrock Edition ที่สร้างโดยผู้พัฒนาเดียวกับ Blockbench และ Snowstorm ซึ่งใช้สำหรับสร้างบทสนทนาของ NPC (NPC dialogues) ครับ

การสร้างสกินและ Resource Packs

การสร้างสกินและ Resource Pack ส่วนใหญ่สามารถทำได้ในโปรแกรมแต่งภาพทั่วไป แต่ก็ยังมีแอปพลิเคชันตัวช่วยบางตัวที่จะทำให้ขั้นตอนนี้ง่ายขึ้นครับ

Blockbench Skin Editor

Blockbench มีระบบแก้ไขสกินในตัวที่ให้คุณแก้ไขได้ทั้งแบบ 3D (และ 2D) มาพร้อมเครื่องมือระบายสีขั้นสูงของ Blockbench เช่น การวาดแบบกระจกเงา (Mirror painting) และจานสี (Color palettes) นอกจากนี้ยังมีค่าพรีเซ็ตสำหรับสร้าง Texture ของสิ่งมีชีวิต (Entity) สำหรับ Resource Pack ด้วยครับ

และคุณยังสามารถแพ็คสกินเพื่อส่งขึ้น Marketplace ได้โดยใช้ปลั๊กอิน Skin Pack Packager ใน Blockbench ครับ

PMCSkin3D

PMCSkin3D เป็นตัวแก้ไขสกินบนเว็บไซต์จาก PlanetMinecraft มีโหมดทั้ง 3D และ 2D รวมถึงโหมดแบ่งหน้าจอ (Split-screen) และตัวเลือกหัวแปรงขั้นสูง นอกจากนี้ยังมีรุ่น Lite ที่ลดทอนเครื่องมือให้ใช้งานง่ายขึ้นด้วยครับ

Aseprite

Aseprite เป็นแอปพลิเคชันศิลปะแบบเสียเงิน (Purchasable) ที่ออกแบบมาเพื่องาน Pixel-art โดยเฉพาะ มีเครื่องมือมากมายที่ช่วยให้การสร้างสกินและ Resource Pack ง่ายขึ้นมาก

Aseprite ยังมีเอกสารและบทช่วยสอนจำนวนมาก เพื่อช่วยให้ศิลปินทุกระดับทักษะใช้งานโปรแกรมได้อย่างมีประสิทธิภาพครับ

McSkin3d

McSkin3d เป็นซอฟต์แวร์ที่ออกแบบมาเพื่อการสร้างสกินโดยเฉพาะ มีฟีเจอร์มากมาย เช่น ตัวดูสกินในตัว (Built-in skin viewer) ที่สามารถระบายสีลงไปได้โดยตรง และมุมมองแบบแบ่งหน้าจอที่แสดงสกิน 3D ควบคู่ไปกับ Texture 2D ครับ

LookatMySkin

LookatMySkin เป็นโปรแกรมดูตัวอย่างสกินที่ช่วยให้คุณแก้ไขสกินในโปรแกรมโปรดของคุณ แล้วดูตัวอย่างบนโมเดลตัวละคร 3D ได้ทันทีแบบเรียลไทม์

มีฟีเจอร์หลากหลาย เช่น การจัดท่าทางแขนขา (Limb posing), การซูม, และการปรับสีพื้นหลัง ซึ่งมีประโยชน์มากสำหรับการถ่ายภาพหน้าจอ (Screenshots) ของสกิน หรือตรวจสอบความผิดปกติของสกินเมื่อจัดท่าทางต่างๆ ได้อย่างรวดเร็วครับ
