พื้นฐานการเริ่มต้นพัฒนา Add-ons

ไม่ว่าคุณจะอยากสร้างสรรค์ผลงานด้วยตนเอง หรือได้รับแรงบันดาลใจจากเหล่าผู้สร้างคนอื่นๆ นี่คือก้าวแรกสู่การยกระดับการปรับแต่งให้กับ Minecraft: Bedrock Edition

Add-ons ช่วยให้ผู้เล่นสามารถปรับเปลี่ยนรูปลักษณ์ของโลก หรือแม้แต่พฤติกรรมของ Entity อย่างเช่น คุณสามารถเปลี่ยนรัศมีการระเบิดของ Creeper หรือสร้างมอนสเตอร์ชนิดใหม่ขึ้นมาเองได้!

สิ่งที่คุณจะได้เรียนรู้

  • วิธีการดาวน์โหลดและติดตั้ง Add-ons สำหรับอุปกรณ์ต่างๆ
  • โครงสร้างไฟล์และตำแหน่งที่สำคัญ (com.mojang)
  • เครื่องมือที่แนะนำสำหรับเขียนโค้ด

1. การติดตั้ง Add-ons

วิธีการติดตั้งจะแตกต่างกันไปตามอุปกรณ์ที่คุณใช้งาน เลือกหัวข้อด้านล่างให้ตรงกับอุปกรณ์ของคุณ:

  1. ตรวจสอบให้แน่ใจว่าติดตั้ง Minecraft: Bedrock Edition แล้ว
  2. ดาวน์โหลดไฟล์ Add-ons (สกุลไฟล์ .mcpack หรือ .mcaddon)
  3. หากไฟล์ที่ได้เป็น .zip ให้เปลี่ยนนามสกุลเป็น .mcpack หรือ .mcaddon
  4. ดับเบิ้ลคลิกเพื่อเปิดไฟล์ ตัวเกมจะเปิดขึ้นอัตโนมัติ
Tip: หากขึ้นแจ้งเตือนว่า "Successfully imported" แสดงว่าเสร็จสมบูรณ์ อย่าลืมไปเปิดใช้งานใน Resource/Behavior Packs ในการตั้งค่าโลกของคุณ
  1. ติดตั้งแอปจัดการไฟล์ (File Manager) เช่น RS File Explorer หรือใช้ของเครื่อง
  2. ดาวน์โหลดไฟล์ Add-on ลงในเครื่อง
  3. เปลี่ยนนามสกุลไฟล์จาก .zip เป็น .mcpack หรือ .mcaddon
  4. กดเปิดไฟล์และเลือกให้เปิดด้วยแอป Minecraft

ข้อควรระวัง: ตรวจสอบการตั้งค่า Storage ในเกมว่าเป็น "Application" หรือ "External" (แนะนำ External เพื่อจัดการไฟล์ง่ายขึ้น)

  1. เปิดแอป Files บน iOS
  2. ไปที่ไฟล์ที่ดาวน์โหลดมา เปลี่ยนนามสกุลเป็น .mcpack หรือ .mcaddon
  3. แตะที่ไฟล์เพื่อเปิด เกม Minecraft จะเด้งขึ้นมาและ Import อัตโนมัติ

2. รู้จักกับโฟลเดอร์ com.mojang

หัวใจสำคัญของการทำ Add-ons คือโฟลเดอร์ที่ชื่อว่า com.mojang นี่คือที่เก็บไฟล์ทั้งหมด ทั้ง Resource Packs และ Behavior Packs สำหรับ Windows คุณสามารถเข้าถึงได้ง่ายๆ ดังนี้:

กดปุ่ม Win + R แล้วคัดลอกคำสั่งนี้ไปวาง:

%appdata%\Minecraft Bedrock\users\shared\games\com.mojang

ภายในคุณจะพบกับโฟลเดอร์สำคัญ:

  • development_behavior_packs: สำหรับวางโค้ดที่คุณกำลังพัฒนา (เกมจะรีโหลดให้เอง)
  • development_resource_packs: สำหรับวาง Texture/Model ที่คุณกำลังพัฒนา

3. เครื่องมือสำหรับเขียนโค้ด (IDE)

การมีเครื่องมือที่ดีช่วยให้งานเสร็จไวขึ้นและลดข้อผิดพลาด นี่คือ 2 โปรแกรมที่แนะนำ:

Bridge. v2

โปรแกรมที่ออกแบบมาเพื่อ Minecraft Add-ons โดยเฉพาะ มีเทมเพลตสำเร็จรูปและระบบช่วยเขียนโค้ดที่ยอดเยี่ยม เหมาะมากสำหรับผู้เริ่มต้น ใช้งานได้ทั้งบน Web Browser และ Desktop

Visual Studio Code (VS Code)

เครื่องมือระดับมือโปรจาก Microsoft เหมาะสำหรับการเขียนโปรเจกต์ขนาดใหญ่ รองรับ Extension สำหรับ Minecraft Bedrock ที่ช่วยตรวจสอบความถูกต้องของ JSON ได้อย่างแม่นยำ

Extensions แนะนำสำหรับ VS Code

เพื่อการเขียนโค้ดที่ลื่นไหล อย่าลืมติดตั้งส่วนขยายเหล่านี้:

  • Blockception's Minecraft Bedrock Development
    เครื่องมือสารพัดประโยชน์ ช่วยตรวจสอบความถูกต้องของ JSON และมีระบบ Auto-completion ช่วยเดาคำสั่งให้แม่นยำ
  • Minecraft Bedrock Debugger
    เชื่อมต่อ VS Code เข้ากับตัวเกม Minecraft เพื่อดู Error และ Debug สคริปต์ได้แบบ Real-time
  • Minecraft Color Highlighter
    ช่วยแสดงสีของรหัสสีในโค้ด (เช่น §a, §c) ให้เห็นภาพทันที ไม่ต้องนั่งเดาว่าสีไหนเป็นสีไหน
  • Snowstorm
    ตัวช่วยสร้างและแก้ไขไฟล์ Particle Effect ในรูปแบบกราฟิก (Visual Editor) ภายใน VS Code โดยตรง
  • Datapack Icons
    เพิ่มไอคอนหน้าชื่อไฟล์ต่างๆ ให้ดูง่ายขึ้น แยกแยะได้ทันทีว่าไฟล์ไหนเป็น Entity, Item หรือ Block
